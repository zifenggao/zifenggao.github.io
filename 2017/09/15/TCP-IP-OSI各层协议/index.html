转：TCP/IP OSI各层协议，ARP属于哪一层的协议

Ip是不可靠协议

  • 在OSI模型中ARP协议属于链路层；而在TCP/IP模型中，ARP协议属于网络层。
  • ARP报文是由以太网帧进行封装传输的。没有封装进IP包。

HTTP也是应用层协议，SSL(Secure Sockets Layer 安全套接层),及其继任者传输层安全(Transport Layer Security,TLS)是为网络通信提供安全及数据完整性的一种安全协议，工作在传输层

发表于 2017-04-15 | 分类于 Hexo

NexT主题配置进阶

主题配置包括：

  • 设置「RSS」
  • 添加「标签」页面 如何为文章添加标签或者分类
  • 添加「分类」页面
  • 设置「字体」
  • 设置「代码高亮主题」
  • 侧边栏社交链接
  • 开启打赏功能
  • 设置友情链接
  • 腾讯公益404页面
  • 站点建立时间
  • 订阅微信公众号
  • 设置「动画效果」
  • 设置「背景动画」

第三方服务包括：

  • 评论系统
  • 数据统计与分析
  • 内容分享服务
  • 搜索服务
  • 其他服务

稍微写一下基本操作：

标签

  1. 新建标签页面：（这样标签合辑的页面被制作出来）

  2. 编辑刚刚新建的页面，将其页面类型设置为tags（这样主题将知道这是一个标签页面，并为这个页面显示标签云）

  3. 编辑主题配置文件，添加tags到menu目录中

  4. 之后在新建的文章中就可以通过设置文章中的开始的声明来为文章添加标签了
